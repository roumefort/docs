# Documentation R-Type

## Description

This is the documentation for the R-Type project. This project is a 2D game based on the R-Type game. The game is a side-scrolling shoot-em-up game where the player controls a spaceship and has to destroy all the enemies in the level. The player can move the spaceship up and down, and shoot

## Installation

To install the game, you need to download the latest release from the [github repository](https://github.com/EpitechPromo2027/B-CPP-500-PAR-5-2-rtype-mikael.sourati)

launch the compilation and installaion with the following commands:

```bash
./make.sh
```

you can then eitheir launch the server or the client with the following commands:

```bash
./rtype_server
```

```bash
./rtype_client
```

## Usage

The game is a multiplayer game, you can either host a game or join a game. To host a game, you need to launch the server and then launch the client. To join a game, you need to know the IP address of the server and then launch the client.
